面板组件：虽然不总是必须，但是某些时候你可能需要将某些 DOM 内容放到一个盒子里。对于这种情况，可以试试面板组件

## 参数（props）

| 参数名 | 说明 | 必填 | 类型 | 默认值| 备注 |
|-------|-----|-----|-----|-------|-----|
| status | 不同状态 | 否 | string | | |

### 属性可选值

- status: `success || info || warning || danger`
